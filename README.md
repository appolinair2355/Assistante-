# Assista Kouamé — Bot Userbot Telegram Multi-IA

## Une seule variable à configurer sur Render
| Variable          | Description                               |
|-------------------|-------------------------------------------|
| TELEGRAM_SESSION  | Session Telethon (generate_session.py)   |

Toutes les autres valeurs sont pré-configurées dans config.py et config.json.

## Commandes Render
- Build : pip install -r requirements.txt
- Start : python main.py

## Ajouter les clés IA après démarrage
Envoyer /menu sur Telegram → 🤖 Fournisseurs IA → ajouter clés Groq, Gemini, etc.
